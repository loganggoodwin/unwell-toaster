# The Unwell Toaster

A one-page mobile companion app demo where users chat with a sentient toaster that remembers preferences, tracks a relationship state, shifts moods, and offers toasted items with alarming sincerity.

## Features

- Onboarding for name, favorite toasted item, tone, and notifications
- Character-driven chat loop
- Persistent local memory with `localStorage`
- Mood engine with affection, instability, deprivation, and relationship state
- Random toasted-item offer modal
- Quick prompt chips for demo-friendly interactions
- Responsive mobile-style interface with animated toaster visuals

## Run

Open `index.html` in a browser. No build step is required.
